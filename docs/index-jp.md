# UE4SS Mod Manager
## インストール
### インストール方法（ツール本体）
ダウンロードした ZIP ファイルを任意の場所に展開してください。

### インストール方法（UE4SS カスタムビルド）
展開したフォルダ内の `UE4SS_ModManager.exe` を実行してください。  
次の画面が表示されます。

![UE4SS Mod Manager screen 1](https://staticdelivery.nexusmods.com/mods/3232/images/312/312-1772028833-1028311452.png)

赤枠の「Install UE4SS...」ボタンをクリックし、表示されたダイアログから `ToM-UE4SS-Custom` の ZIP ファイルを選択してください。

「Status」欄に「Installed」が緑色で表示されれば、インストール成功です。

### 新しい MOD のインストール
![UE4SS Mod Manager screen 2](https://staticdelivery.nexusmods.com/mods/3232/images/312/312-1772028843-701758782.png)

UE4SS のインストール後、画面下部の「Install MOD...」ボタンをクリックし、表示されたダイアログから UE4SS 用 MOD の ZIP ファイルを選択することで、新しい MOD をインストールできます。

## アンインストール
### MOD のアンインストール
アンインストールしたい MOD のチェックボックスをオンにし、画面下部の「Delete Selected」ボタンをクリックしてください。MOD は完全に削除されます。

削除した MOD を再度使いたい場合は、「新しい MOD のインストール」セクションの手順に従って再インストールしてください。

### UE4SS のアンインストール
UE4SS Status 欄の「Uninstall」ボタンをクリックしてください。確認ダイアログが表示されるので、OK を押すと UE4SS がアンインストールされます。

UE4SS を再インストールしたい場合は、「インストール方法（UE4SS カスタムビルド）」セクションの手順に従ってください。

### ツール本体のアンインストール
ツールの exe ファイルが入っているフォルダごと削除してください。

## その他の機能
### MOD の無効化 / 有効化
インストール済み MOD を一時的に無効化または有効化できます。MODを無効化することで、ファイルを保持したまま、その MOD が未インストールであるかのようにゲームをプレイできます。

**MOD を無効化する方法:** 無効化したい MOD のチェックボックスをオンにし、画面下部の「Disable Selected」ボタンをクリックしてください。MOD の「Status」列が「Disabled」と表示されれば無効化完了です。

**MOD を有効化する方法:** 有効化したい MOD のチェックボックスをオンにし、画面下部の「Enable Selected」ボタンをクリックしてください。MOD の「Status」列が「Enabled」と表示されれば有効化完了です。

## トラブルシューティング
ゲームが管理者権限を必要とするフォルダにインストールされている場合、UE4SS や MOD のインストールに失敗することがあります。  
その場合は、ゲームが終了していることを確認し、UE4SS Mod Manager を管理者権限で再起動してから再度お試しください。

ツールがゲームフォルダのパスを正しく検出できていない場合(`Game Folder Path`に表示されているパスにゲームがインストールされていない場合)、
UE4SSが正しくインストールできません。
その場合は、`Game Folder Path`が表示されているテキストボックスの右側にある「Browse」ボタンを押して、
表示されるダイアログから正しいパスを手動で設定してください。

その他、ツールが動作しない場合は以下のNexusModsのページでお問い合わせください。
https://www.nexusmods.com/trialsofmana/mods/312?tab=posts
