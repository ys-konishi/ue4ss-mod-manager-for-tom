## UE4SS Mod Manager (Recommended, Update Mar 1, 2026)

I provide a tool called UE4SS Mod Manager, which supports installing and uninstalling UE4SS and mods for UE4SS.

Extract the downloaded UE4SS_ModManager-1.0.1.zip file and run UE4SS_ModManager.exe inside the extracted folder.

The following screen will appear.

---

UE4SS本体およびUE4SS用のMODのインストール・アンインストールをサポートするツール(UE4SS Mod Manager)を提供しています。  
ZIPファイルを解凍したフォルダ内にあるexeファイルをクリックすると次のような画面が表示されますので、"Install UE4SS.." ボタンを押して、表示されるダイアログからToM-UE4SS-CustomのZIPファイルを選択してください。

![UE4SS Mod Manager screen 1](https://staticdelivery.nexusmods.com/mods/3232/images/312/312-1772028833-1028311452.png)

Click the red-framed "Install UE4SS..." button, then select the ToM-UE4SS-Custom zip file from the dialog that appears.

If the word "Installed" appears in green text in the "Status" frame, the installation has been successful.

![UE4SS Mod Manager screen 2](https://staticdelivery.nexusmods.com/mods/3232/images/312/312-1772028843-701758782.png)

After installing UE4SS, you can install a new MOD by clicking the "Install MOD..." button at the bottom of the screen  
and selecting the UE4SS MOD ZIP file from the dialog that appears.

---

UE4SS本体をインストール後、画面下部の "Install MOD..." ボタンを押して表示されるダイアログから  UE4SS用MODのZIPファイルを選択することで、新しいMODをインストールできます。

## Trouble Shoot

If the game is installed in a folder that requires administrator privileges, the installation of UE4SS or MODs may fail.  
In that case, make sure the game is closed, restart the UE4SS Mod Manager with administrator privileges, and try again.

---

ゲームが管理者権限が必要なフォルダにインストールされている場合、UE4SS本体やMODのインストールに失敗する場合があります。  
その場合はゲームが終了していることを確認し、UE4SS Mod Managerを管理者権限で再起動してやり直してください。
