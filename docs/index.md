# UE4SS Mod Manager
## Installation
### How To Install (Tool itself)
Please extract the downloaded ZIP file to any location you prefer.

### How To Install (UE4SS Custom Build)
Please run UE4SS_ModManager.exe inside the extracted folder.
The following screen will appear.

![UE4SS Mod Manager screen 1](https://staticdelivery.nexusmods.com/mods/3232/images/312/312-1772028833-1028311452.png)

Click the red-framed "Install UE4SS..." button, then select the ToM-UE4SS-Custom zip file from the dialog that appears.

If the word "Installed" appears in green text in the "Status" frame, the installation has been successful.

### Install New MODs
![UE4SS Mod Manager screen 2](https://staticdelivery.nexusmods.com/mods/3232/images/312/312-1772028843-701758782.png)

After installing UE4SS, you can install a new MOD by clicking the "Install MOD..." button at the bottom of the screen and selecting the UE4SS MOD ZIP file from the dialog that appears.

## Uninstall
### Uninstall MODs
Check the checkbox next to the MOD you want to uninstall, then click the "Delete Selected" button at the bottom of the screen. The MOD will be completely removed.

If you want to use an uninstalled MOD again in the future, you will need to follow the steps described in the "Install New MODs" section to reinstall it.

### Uninstall UE4SS
Click the "Uninstall" button in the UE4SS Status box. A confirmation dialog will appear. Click OK to uninstall UE4SS.

If you want to reinstall UE4SS in the future, follow the steps described in the "How To Install (UE4SS Custom Build)" section.

### Uninstall Tool itself
Simply delete the entire folder containing the tool's exe file.

## Other functions
### Disable / Enable MODs
You can temporarily disable or enable installed MODs. When you disable a MOD, the files are retained, but the game will run as if the MOD is not installed.

**To Disable a MOD:** Check the checkbox next to the MOD you want to disable, then click the "Disable Selected" button at the bottom of the screen. Once the MOD's "Status" column shows "Disabled", the MOD is disabled.

**To Enable a MOD:** Check the checkbox next to the MOD you want to enable, then click the "Enable Selected" button at the bottom of the screen. Once the MOD's "Status" column shows "Enabled", the MOD is enabled.

## Trouble Shoot
If the game is installed in a folder that requires administrator privileges, the installation of UE4SS or MODs may fail.  
In that case, make sure the game is closed, restart the UE4SS Mod Manager with administrator privileges, and try again.
